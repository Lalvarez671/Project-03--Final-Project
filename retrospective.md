# Project Retrospective

# What Went Well 
One thing that went well during this project was organizing the professional GitHub repository and connecting the  HTML and CSS files correctly. I also gained more confidence with using basic web development tools.

## What Went Wrong
One challenge I faced was managing time while balancing school and personal responsibilities. There were also moments where troubleshooting HTML and CSS errors became frustrating.

## What I Would Do Differently
If I completed this project again, I would begin designing earlier and test the website more often throughout development. I would also spend more time improving the visual design and responsiveness of the website.
